Overview:

I intend to code an algorithm that uses the Newton-Raphson method to calculate the roots of a function. I’d also like to include a visual component that opens a window in which the program graphs the function and the tangent lines which are used to approximate the root.

Method:

The Newton-Raphson method is defined on wikipedia as:

“a root-finding algorithm which produces successively better approximations to the roots (or zeroes) of a real-valued function. The most basic version starts with a real-valued function \({f}{}\), its derivative \({f'}{}\), and an initial guess \({x_0}\) for a root of \({f}{}\). If \({f}{}\) satisfies certain assumptions and the initial guess is close, then: \(x_1 = x_0 - \frac{f(x_0)}{f'(x_0)}\)"

How I’ll do it:

This will likely be done with mostly Python in Pycharm or Google Collab (if the program ends up being computationally light.) I plan to do the computation with SymPy, as I have practice with it from the Explorations. The graphical part will likely also be with Sympy, but MatPlotLib is my backup, thanks to my brief experience with it in Scientific Computing.
